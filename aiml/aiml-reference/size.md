### &lt;size /&gt;

The *size element* returns the number of category blocks contained in the current bot.

#### Usage

    <category>
      <pattern>TEST SIZE</pattern>
      <template><size/></template>
    </category>

If your bot has 1000 categories:

>**Input:** test size  
**Output:** 1000
