### &lt;aiml&gt;

The *AIML root element* delimits a block of AIML code. All other elements must be descendents of the root element.

#### Attributes

`version` (optional)
Specifies the AIML version that the document is written in.

#### Usage

    <?xml version="1.0" encoding="UTF-8"?>
    <aiml version="2.0">
    <!-- AIML code goes here -->
    </aiml>
