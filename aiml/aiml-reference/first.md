### &lt;first&gt;

The *first element* returns the first word found in its contents. This is an implementation of list processing in AIML.

#### Usage

    <category>
    <pattern>FIRST *</pattern>
    <template><first><star /></first></template>
    </category>

>**Input:** First a b c d  
**Output: a**
